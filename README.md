# endorsement-network

## Prerequisites
- Node.js (v18+ recommended)
- MongoDB (local or Atlas)
- Git

---

## 1. Clone Repository
```bash
git clone https://github.com/Sandeep-Chagapuram/endorsement-network.git
cd endorsement-network

```bash
cd backend
npm install
```
Create a .env file inside backend/:
```bash
PORT=5001
MONGODB_URI=mongodb://localhost:27017/endorsement-network
```
```bash
npm run dev or node server.js
```

```bash
cd ../frontend
npm install
npm run dev
